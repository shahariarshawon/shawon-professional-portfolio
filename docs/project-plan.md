# Project Plan

## Project Name
Shawon Portfolio

## Main Goal
Build a professional, dynamic, backend-focused portfolio website with admin dashboard.

## Architecture
Separate frontend and backend.

## Folders
- client: Next.js frontend
- server: Express.js backend

## Main Sections
1. Navbar
2. Hero
3. About
4. Experience
5. Skills
6. Projects
7. Education and Certifications
8. Services
9. Contact
10. Footer

## Admin Features
- Login
- Dashboard overview
- Hero manager
- About manager
- Experience manager
- Skills manager
- Project manager
- Education and certification manager
- Services manager
- Contact manager
- Resume builder
- Site settings
